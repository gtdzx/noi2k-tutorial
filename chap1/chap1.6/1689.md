# 信息
题目：《数组逆序重放》
链接：http://noi.openjudge.cn/ch0106/04/
作者：
时间：2014-12-30  


完整程序:
```cpp
#include <iostream>
#include <cstdio>
using namespace std;
int a[200];
int main()
{
	int n;
	scanf("%d",&n);
	for(int i = 0;i < n; ++i)
		scanf("%d",&a[n-i-1]);
	for(int i = 0;i < n; ++i)
		printf("%d ",a[i]);
	return 0;
}
```
