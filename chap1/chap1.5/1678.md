# 信息
题目：《整数的个数》
链接：http://noi.openjudge.cn/ch0105/09/
作者：
时间：2014-12-30  


完整程序:
```cpp
#include <iostream>
#include <cstdio>
using namespace std;
int main()
{
	int k;
	scanf("%d",&k);
	int times1 = 0,times5 = 0,times10 = 0;
	for(int i = 0;i < k ; ++i) {
		int a;
		scanf("%d",& a);
		if( a == 1)
			++times1;
		else if( a == 5)
			++times5;
		else if( a == 10)
			++ times10;
		
	}
	printf("%d\n",times1);
	printf("%d\n",times5);
	printf("%d\n",times10);
	return 0;
}

```
