# 信息
题目：《数1的个数》
链接：http://noi.openjudge.cn/ch0105/35/
作者：
时间：2014-12-30  


完整程序:
```cpp
#include <iostream>
#include <cstdio>
using namespace std;
int main()
{
	int n;
	
	scanf("%d",&n);
	int sum = 0;
	for(int i = 1;i <= n; ++i)	 {
		int k = i;
		while(k >0) {
			if( (k%10) == 1)
				++sum;
			k/= 10;
		}
	}
	printf("%d",sum);
	return 0;
}
```
