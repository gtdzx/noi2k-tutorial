# 信息
题目：《与7无关的数》
链接：http://noi.openjudge.cn/ch0105/34/
作者：
时间：2014-12-30  


完整程序:
```cpp
#include <iostream>
#include <cstdio>
using namespace std;
int main()
{
	int n;
	scanf("%d",&n);
	int sum = 0;
	for(int i = 1; i<= n; ++i) {
		if( i % 7) {
			int k = i;
			while(k>0) {
				if( ( k % 10) == 7)
					break;
				else 
					k /= 10;
			}
			if( k == 0)
				sum += i * i;
		}
	}
	printf("%d",sum);
	return 0;
}
```
