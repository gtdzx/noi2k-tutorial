# 信息
题目：《求阶乘的和》
链接：http://noi.openjudge.cn/ch0105/31/
作者：
时间：2014-12-30  


完整程序:
```cpp
#include <iostream>
#include <cstdio>
using namespace std;
int main()
{
	int n;
	scanf("%d",&n);
	int sum = 0;
	int f = 1;
	for(int i = 1;i <= n; ++i) {
		f *= i;
		sum += f ;
		
	}
	printf("%d\n",sum);
	return 0;
}
```
